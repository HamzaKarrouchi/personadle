📋 Classic Mode Rules

🎯 Goal:
Guess the Persona character of the day using deduction and clues 🧩🕵️.

🧠 How it works:
  • 🔍 Each attempt shows how close your guess is to the correct character.
  • 📊 A grid compares multiple attributes like name, gender, age, arcana, etc.
  • 🟩 Green = correct, 🟨 Yellow = partial match, 🟥 Red = wrong.
📊 Criteria compared:
  • 👤 Name – Exact match required
  • ⚧️ Gender – Human, Shadow, Entity, etc.
  • 🎂 Age – Range (e.g. <15, 15–20, 40+)
  • 💥 Persona User / Shadow – Can they use a Persona or are they a Shadow?
  • 👺 Persona – The specific Persona or Shadow form
  • 🃏 Arcana – Their Tarot Arcana (Fool, Magician, etc.)
  • 🎮 Opus – Which Persona games they appear in
💡 Tips:
  • 🧠 Use deduction based on color-coded clues.
  • 📈 If the age isn’t exact, an arrow ↑ or ↓ appears.
  • 🟨 Yellow means partial match (e.g. some Arcana or one of several Opus).
🎮 Game flow:
  • ⌨️ Type a name and select from suggestions.
  • 🔁 You have 8 attempts max.
  • 💬 At 3 attempts: 🔓 Hint button unlocks (displays a quote).
  • ❌ At 8 attempts: 🔓 Give Up button unlocks (reveals the answer).
🛡️ Spoiler protection:

🕹️ Use the filter buttons (P1, P2, P3, P4, P5) to limit characters by game.
👀 This helps avoid spoilers if you haven’t played them all!
