Classic Mode Rules

🎯 Goal:
Guess today’s Persona character by entering their full name or alias. Each attempt fills a row with the following attributes:

  • 🧍 Name — The character's full name as known in the database.
  • 🚻 Gender — Can be: Male, Female, Artificial, Entity, Shadow, Velvet.
  • 🎂 Age — Value ranges: < 15, 15-20, 21-40, 40+, 80+.
  • 🎭 Persona User / Shadow — Shows Yes if the character is a Persona user or a Shadow. Otherwise: No.
  • 👻 Persona / Shadow — The exact name of their Persona or shadow form. Example: Arsène, Thanatos, Lucia, or NONE.
  • 🃏 Arcana — Associated tarot Arcana. Examples: Fool, Lovers, Death, Magician.
  • 🎮 Opus — Games in which the character appears: P1, P2IS, P2EP, P3, P3FES, P3P, P4, P4G, P4AU, P4D, P5, P5R, P5S, P5T,
🎨 Feedback colors:
  • Green — Exact match.
  • Yellow — Partially correct (e.g. right Arcana but not all values match).
  • Red — Completely incorrect.
  • ↑ / ↓ — For age: indicates if your guess is younger or older than the target.
🧠 Hint system:
- After 3 attempts: a quote can be revealed.
- After 8 attempts: the Give Up button appears to reveal the answer.
