Emoji Mode Rules

🎯 Goal:
Guess which Persona character is being described by a sequence of emojis.

🧠 How it works:
  • You start with 1 emoji.
  • Each wrong guess unlocks 1 more emoji.
  • Once correct, the full emoji chain is revealed.
💡 Tips:
  • Emojis may refer to: personality, arcana, story events, role, weapon...
  • Example: 🗡️⚡😐 might describe a serious Persona user with a sword and electric element.
🎮 Game flow:
  • Type a character’s name and choose from suggestions.
  • Wrong answers = more clues. 8 tries max.
  • You can give up to see the correct answer and emojis.
🛡️ Spoiler protection:

Use the P3 / P4 / P5 filter buttons to limit which games the characters are drawn from. This helps avoid spoilers if you haven’t played every game yet.
