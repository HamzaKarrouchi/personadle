Emoji Mode Rules

🎯 Goal:
Guess which Persona character is being described by a sequence of emojis.

  • You start with 1 emoji.
  • Each wrong guess unlocks 1 new emoji.
  • Once the correct answer is found, all emojis are revealed.
🧠 Tips:
  • Emojis can reflect the character’s role, personality, arcana, events, or abilities.
  • Example: 🗡️⚡😐 could represent a serious sword-wielding Persona user.
🎮 Game flow:
  • Type a name and select it from the autocomplete suggestions.
  • If the guess is wrong, you get more visual clues (emojis).
  • If you give up, the answer and full emoji chain are revealed.
